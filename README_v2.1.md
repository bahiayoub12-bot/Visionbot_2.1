## 🧠 نظام الذاكرة

VisionBot يحفظ كل نقرة ناجحة في `vision_memory.json`:
- بعد **3 نجاحات** → يثق بالإحداثيات ويتخطى الـ API
- **متوسط متحرك** للإحداثيات للدقة المستمرة
- **إحصائيات** عبر زر "عرض الذاكرة"

---

## ⑦ Pre-Click Verification

قبل كل نقر:
1. تحرك الماوس للموقع
2. التقاط 200×200 بكسل حول المؤشر
3. سؤال LLM: "هل هذا هو العنصر الصحيح؟"
4. إذا نعم → انقر | إذا لا → أعد الحساب

**يمنع الأخطاء الفادحة** كإغلاق ملف بدل حفظه.

---

## ⑥ ACTION FORMAT الموحد

أي نموذج يُعيد نفس الصيغة:
```json
{
  "action": "CLICK",
  "x": 500,
  "y": 300,
  "confidence": 0.92,
  "element": "زر الإرسال",
  "reason": "هذا الزر سيرسل الرسالة"
}
```
**تغيير المزود لا يحتاج تغيير أي كود.**

## ⑪ المراقب الصامت

```python
rules = [
    {
        "condition": "هل يوجد إعلان أو نافذة منبثقة؟",
        "action": lambda: close_popup()
    },
    {
        "condition": "هل اكتمل التحميل؟",
        "action": lambda: notify("انتهى!")
    }
]
agent.start_watcher(rules)
```
